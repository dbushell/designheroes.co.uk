The Lost Type Co-op have released four new fonts. Mensch is particularly nice!

It's great to see these micro-foundries (I just invented that term, like "micro-brewery") release such high quality fonts. My personal font design project is still a few years from release, mostly because I haven't had time, but more importantly, creative a typeface is really bloody difficult!
