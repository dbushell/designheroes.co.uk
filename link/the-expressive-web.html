Review

The Expressive Web

Posted by David Bushell

The Expressive Web is a collaboration between Adobe and Big Spaceship (a New York agency) showcasing the wonders of web standards.

In addition to highlighting and providing information on twelve new HTML5 and CSS3 features, the site itself makes extensive use of new features such as CSS3 transitions, CSS3 transforms, web storage and more to provide a visually compelling resource for learning more about HTML5 and CSS3.

It's nice to see another tech giant get behind web standards, we've aready seen similar projects by Mozilla, Apple and friends. With the release of Adobe Edge (my Edge review) are we seeing a dramitic move away from Flash?
