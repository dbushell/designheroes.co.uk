Review

978 Grid System

Posted by David Bushell

The 978 Grid System is a fixed-width but responsive grid for website design. The grid was originally created by Nick La while the promotional website is by Brothers Roloff.

The 978 grid system uses 12 columns at 54px, with extra roomy 30px gutters. This comes out to a total of 978px, which uses up most of the space on a 1024x768 monitor. Almost every computer and tablet in the world today is capable of displaying designs built on this grid.

The system also includes the use of CSS media queries to provide resized grids for 748 and 300 pixel widths. It's not fluid but it is responsive for modern devices.

With all the varying grid systems suggested it is important to remember that these are just tools. There is no "right" solution for every website design.
