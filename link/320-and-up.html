320 and up

Posted by David Bushell

320 and up is a ‘tiny screen first’ boilerplate extension made by Andy Clarke.

‘320 and Up’ prevents mobile devices from downloading desktop assets by using a tiny screen’s stylesheet as its starting point. Try this page at different window sizes and on different devices to see it in action.

Read Andy's blog post for more details. I can see this being a great practice for simple content websites where nothing needs to be hidden. It not a universal solution (there never will be) but you should add this technique to your tool belt.
