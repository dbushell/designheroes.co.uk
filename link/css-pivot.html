CSS Pivot

Posted by David Bushell

CSS Pivot lets you add CSS styles to any website, and share the result with a short link.

I can see this being a useful little resource, especially when explaining how to fix little CSS bugs to other people. A sign-in with Twitter would be helpful for keeping track of 'pivots', though there are plenty of options with Google and OpenID.
