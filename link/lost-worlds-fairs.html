Review

Lost World's Fairs

Posted by David Bushell

Lost World's Fairs is a celebration of web typography created by design agency Mighty and friends. The project is lead by web design superstar Jason Santa Maria.

Internet Explorer 9 now supports WOFF, and the Friends of Mighty have joined forces to explore typographic possibilities on the web.

The website presents a small collection of beautifully illustrated pages that make heavy use of web fonts and typographic design. For me "The Moon" stands out on top, but all three designs currently on display are amazing achievements.

You can read more about the project on the authors various blogs, see the site, but here's an interesting quote:

We set out to design first for IE9, so we resolved not to use any properties that weren’t supported yet. Even with that said, IE9 is a really polished browser, and the improved type rendering alone makes it a big step forward for Windows.

Type rendering is still a major hurdle (perhaps the last) that is slowing down web typography so it's good to see the Internet Explorer team actively working on this.
