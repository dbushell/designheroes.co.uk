Symbolicode

Posted by David Bushell

Symbolicode makes it easy to find the entity references used to display math & currency symbols, Greek characters, URL Encodings, & punctuation.

It's been a while since I posted a neat little "web app", or "tool", whatever you choose to call it. This one basically replaces copypastecharacter.com which I've been using religiously to copy the © symbol onto my clipboard. Symbolicode doesn't actually allow copying as plain text, which is annoying for Photoshop mockups, but as a tool for HTML is simple and excellent.
