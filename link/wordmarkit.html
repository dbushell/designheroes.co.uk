Review

wordmark.it

Posted by David Bushell

wordmark.it is a simple typography web app by Fahri Özkaramanlı that allows you to quickly view a word in multiple fonts. But don't let the simplicity fool you, this tool is very clever:

I don't know if it's just me but I find myself skimming through my font library every time I am about to design a new wordmark. And the fonts that I end up using are mostly the ones that are already activated/installed on the system.

The site works by detecting the fonts you have already installed (using a bit of Flash font detection magic). Give Fahri Özkaramanlı a shout-out on twitter @fahrio for this time saving website - thank you! A true Design Hero :)
