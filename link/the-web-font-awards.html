Review

The Web Font Awards

Posted by David Bushell

"The Web Font Awards is the first ceremony to celebrate the newfound typographic freedom that Web designers are experiencing across the globe."

Now I wouldn't normally feature something that's already been on every design blog from Creative Review to Swissmiss – you've probably already seen it – but considering I'm riding high at #1 on the community vote right now, I figure I can break my own rules.

If you have time give my website an honest scoring :) The deadline is the 7th November so I don't think I'll stay #1 for very long. I know of plenty of web designers out there I'm just hoping don't have time to enter!

The whole competition is a marketing device by Monotype Imaging to promote the use of web fonts, specifically their web fonts. I'm all for improving typography on the web so it gets a thumbs up from me.
