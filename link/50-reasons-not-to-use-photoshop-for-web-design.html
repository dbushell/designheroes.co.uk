50 reasons NOT to use Photoshop for Web Design

Posted by David Bushell

50 reasons NOT to use Photoshop for Web Design is an article examining the use of Adobe Fireworks and Photoshop for web design, by André Reinegger.

First things first, I love Photoshop. It is the world’s best program for image editing. I do not intend to say Photoshop is a bad program, I just wish to clear up the misunderstanding that Photoshop is the right tool to use for web and screen design.

This is a fascination article and raising some great comparisons.

I use Photoshop to design websites, that's how I work. I've tried Fireworks but it was completely foreign to me. Each to their own I say. It's not the tool but what you do with them!
