Shape Catcher

Posted by David Bushell

Shape Catcher is a drawing canvas that employs character recognition for 10007 unicode glyphs, developed by Benjamin Milde.

This is a tool to help you find Unicode characters. Finding a specific character whose name you don't know is cumbersome. On shapecatcher.com, all you need to know is the shape of the character!

Unicode glyphs are awesome for quick UI icons. I can see myself drawing a lot of arrows!
