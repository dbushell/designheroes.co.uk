The 997 Grid System

Posted by David Bushell

Designing and building websites can be a tricky task. Like our predecessors in print, many web designers will choose an underlying grid to position design elements. Some will take this one step further, creating complex CSS to convert a design into HTML, while retaining the grid as a structure.

Harry Roberts has perfected this art with the 997 Grid System.

Why waste all that valuable screen-space on gutters? Why not just have bigger columns?

Pure genius.

I'm a big fan of designing with a grid (you'll be pleased to know). But when it comes to converting a design into CSS & HTML, trying to replicate a grid with the 960 system or its brethren is perhaps not the best solution for a semantic web. Might as well use tables. Instead master the box model, create a clean, lean, bespoke build for every design.
