Review

Gridness

Posted by David Bushell

Gridness is a graphic design showcase curated by Hazen Creative and dedicated to the grid. If you're like me, you'll start any graphic or web design project with a grid. It automatically gives structure to a page. The daunting empty whitespace becomes an art board for creativity. On this subject Gridness says:

Most often, the grid is what separates real designers from amateurs. And like any tool, it takes awhile to master. It's a visual system developed specifically for the task at hand, in order to arrange the type and imagery in a deliberate way.

Well said indeed.

And the beauty of the grid is knowing when to let it go. Break out of the structure in the right place and your design can be memorizing. It's a difficult beast to tame though, so why not get some inspiration from a site like Gridness? Make sure you study each design to really understand how the grid has been used. You will learn a lot that way.
