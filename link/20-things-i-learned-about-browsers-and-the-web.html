20 Things I Learned About Browsers And The Web

Posted by David Bushell

20 Things I Learned About Browsers And The Web is an Interactive book by Google and Illustrated by Christoph Niemann.

What’s a cookie? How do I protect myself on the web? And most importantly: What happens if a truck runs over my laptop?

A beautiful example of interactive design using web standards and a great education tool for the uninformed. Massive props to Google for this one!
