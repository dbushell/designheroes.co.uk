Google Fonts Directory and API

CSS3 and its font embedding posibilites are exploding glorious typography all over the web. Google have now opened up their APIs and content delivery network to provide hosting for various free fonts. "The Google Font Directory provides high-quality web fonts that you can include in your pages using the Google Font API". Is it too soon to say adios to Arial and co?
