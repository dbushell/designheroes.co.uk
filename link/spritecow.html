SpriteCow

Posted by David Bushell

"Sprite Cow helps you get the background-position, width and height of sprites within a spritesheet as a nice bit of copyable css."

This probably won't make much sense unless you're a front-end web developer. If you're just that, Sprite Cow is magic.

The tool requires support for some very modern web standards and APIs. The "sorry you need Chrome/Firefox" screen includes a 4 stage pass/fail overview, which is so much better than the dreaded "requires browser X" message. Nice work The Team.
