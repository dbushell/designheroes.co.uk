textorize: Pristine Font rendering for the Web

"textorize" is a clever command line script (written in Ruby). It lets Mac OS X users render typography using OS X's subpixel antialiasing and outputs a PNG (by default). The alternative would be using Photoshop various text rendering options which aren't as good in my opinion.
